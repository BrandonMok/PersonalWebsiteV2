Cynergy - Group

When assigned a group project, there are various methods of communication, file sharing, version control, and task distribution. All of which takes time just to simply establish the methods to use as each member has to have an account for each application, and prior knowledge on using a given app. What happens if this group frustration could be eliminated entirely so that the group can focus on what's more important?


Cynergy is the solution to all group assignments! Cynergy is a prototype from the Designing the User Experience course where groups developed their ideas to make a prototype. My group's idea originated from our prior group assignments as simply establishing a common basis for applications took valuable time that could've been spent on the assignment itself; therefore, my group and I devised this idea to hopefully solve this problem.


The project consists of all group necessities, but combined into one application. Our intent is so that our application will replace having the hassle to constantly switch between different apps; therefore, maximizing productivity. Cynergy displays all information such as chat, tasks, and a calendar for the user to have control of everything necessary in a group environment. The project was created in InVision, a UX/UI software.
