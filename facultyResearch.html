Faculty Research - Group

As part of the Database Connectivity & Access course, groups were assigned the task to allow different types of users to search for current faculty research projects.


My group used Java and MYSQL as the user interface was done with JavaFX and the program's functionality of interacting with the database was completed with various classes: presentation layer, business layer, and database layer.


Depending on the type of user when logging in, different functions were available. The available functions consist of select, insert, update, and delete. As a result, a faculty member has access to all functions while a student and public user only have a search function (select).


I primarily fulfilled the front-end of the project as each function has its own screen. For example, if a user logins successfully, the login screen switches to the default search screen. The switching of screens is consistent with all other functions due to each process requiring different types of information to successfully perform the action.


I was also responsible for verifying that my code successfully interacted with the backend code. I troubleshot the information my classes were sending, the variables in the database, and the methods my group created.

Skills Used:

  1. JAVA
  2. MYSQL
