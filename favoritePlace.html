Favorite Place - Individual

Based on the Web & Mobile II course, the project required the creation of an informative website about a favorite place or hometown. Originating from a small town that is 1 mile in size, I didn't have a lot of interesting information to provide; therefore, I chose a favorite place.


The project was relatively large as it consisted of two major parts. The first part was creating the project itself and creating the content while the second was during the last half of the semester where we were to include different technologies learned during the course. The project consists of 17+ pages, a javascript slideshow, PHP form data collection and verification, a connection to PHPMyAdmin, and a comments section.

Skills Used:

  1. HTML
  2. CSS
  3. JS
  4. PHP
