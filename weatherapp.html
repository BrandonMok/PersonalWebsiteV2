React - Weather App

The project acted as my first learning experience of the React javascript library as I followed along a YouTube tutorial by Hamza Mirza. Through the tutorial, I gained an understanding of concepts such as components, props, states, and more!

The weather app provides information about the weather of a given place. It utilizes an api from openweathermap.org for gathering and providing information to the user. A downfall, however, of the app is that only a city and country is accepted: there are multiple cities with the same name in the same country. The api only accepts these two fields as the functionality is there, but only restricted to the first location.

Although I'm still learning, I've gained an interest in the library as I'm always curious in learning more. I've done outside research mainly through Youtube videos, documentation, and other searched topics.
